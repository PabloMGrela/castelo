# Logopedia Castelo

Website for Logopedia Castelo - A speech therapy clinic in A Coruña, Spain.

## About the Clinic

Logopedia Castelo is a speech therapy clinic dedicated to providing professional care across all life stages. Our approach is built on three fundamental principles:

- **Comunicación** (Communication): Essential skills, diversity, interaction, and coordination with families and other professionals
- **Cercanía** (Closeness): Personal treatment, clarity, support, and accessible location
- **Compromiso** (Commitment): Professionalism, continuous training, professional network in A Coruña, and essential family participation

## Services

- **Child, Youth, and Adult Speech Therapy**: From early stimulation to acquired conditions affecting breathing, swallowing, or communication
- **Elderly Adult Care**: Preventive stimulation to mitigate cognitive decline and environmental support strategies
- **Family Intervention**: Resources and support for child tutors and elderly caregivers, addressing emotional burden

## Development

This is a Flutter web application. For development setup:

1. Install Flutter SDK (version 3.9.2 or higher)
2. Run `flutter pub get` to install dependencies
3. Run `flutter run -d chrome` for development
4. Run `flutter build web --release` to build for production
